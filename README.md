# Chat_App | [Link to APK](https://drive.google.com/file/d/1Zhz2wNAEP5qBkh-MsL-vnBBwvQRSK1ym/view?usp=sharing)

Real time chat application designed to facilitate one to one conversation between 2 users. 

### Technologies Used:
* Java
* XML 
* Firebase
* Android Studios
